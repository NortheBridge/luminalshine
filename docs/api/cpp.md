# C++ API Reference

The full Doxygen-generated C++ API browser — class hierarchies, file
listings, namespaces, call graphs, and the complete symbol index — is
served alongside this site:

<a href="../api/index.html"><strong>→ Browse the C++ API at <code>/api/</code></strong></a>

It is regenerated on every Read the Docs build from
[`src/`](https://github.com/NortheBridge/luminalshine/tree/main/src).
The browser uses Doxygen's HTML output directly so the full
configuration in [`docs/Doxyfile`](../Doxyfile) — including custom
JavaScript-driven API examples, search index, and call graphs — is
preserved as-is from the previous documentation pipeline.
